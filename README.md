# ConsoleEx
This is a new Application Console definition.
It is not a Console Calculator Application (this is just an example)
This library definition allow you to modify C# Console application into something else.

## Features
- 16 million possible colors
- 26 customizable colors
- Real-time Mouse and Keyboard Events
- Customized Form with blinking buttons (when Mouse Clicked)
- Exporting/Importing previous defined form

## Features coming soon (in progress)...
- Form input fields
- Fields autocompletion (using Tab key)
	
## Possible future features
- Form on Form
- Ansi Animation
- Scripting
